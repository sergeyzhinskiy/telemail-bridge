Что это?
TeleMail Bridge позволяет получать и отправлять личные сообщения Telegram через обычную электронную почту.

Когда Telegram заблокирован в вашей стране, но email работает (а email блокировать невозможно — это базовая инфраструктура интернета), этот бот становится вашим мостом в Telegram.

Ключевые точки:

Бот крутится на сервере за границей, где Telegram не заблокирован.

Пользователь в РФ заходит только в свой email-клиент — это может быть что угодно: Gmail, Mail.ru, Яндекс.Почта, Thunderbird, Apple Mail, хоть Pine.

Telegram не используется для общения — только почта.

Бот слушает личные сообщения пользователя в Telegram и ретранслирует их на email, и наоборот — ответы с email публикует в Telegram от имени пользователя.

Сценарий использования
Регистрация (один раз):

Пользователь находит способ запустить Telegram (VPN, поездка за границу, прокси).

Запускает бота @TeleMailBot.

Бот просит email и пароль для IMAP/SMTP.

Бот просит авторизоваться в Telegram (код из SMS) — создаёт сессию пользователя.

Бот сохраняет сессию и начинает слушать личные сообщения.

Повседневное использование (в РФ, без Telegram):

Друг пишет пользователю в Telegram: "Привет, как дела?"

Бот видит это сообщение и отправляет пользователю email:

text
От: bot@telemail.app
Тема: Telegram: Иван (@ivan_ru) [ID: 12345]
Тело: Привет, как дела?
Пользователь получает обычный email-пуш на телефон.

Отвечает на письмо: "Привет! Всё хорошо, как у тебя?"

Бот получает ответ, находит чат с Иваном и отправляет ему сообщение в Telegram.

Иван видит обычное сообщение от пользователя.

Результат: полноценное общение, без установки дополнительных приложений, через стандартную почту.

Технологический стек

Компонент	Технология

Telegram Client (слушает сообщения пользователя)	Telethon (User API)
Email-отправка (письма пользователю)	aiosmtplib
Email-получение (ответы пользователя)	aioimaplib + catch-all ящик
Фреймворк бота (регистрация, настройки)	Aiogram 3.x
Админка	FastAPI + Jinja2 + Bootstrap 5
База данных	PostgreSQL + SQLAlchemy + Alembic
Очереди задач	Celery + Redis
Платежи	ЮKassa / Telegram Stars / Stripe
Деплой	Docker + Docker Compose + Nginx
